# graph-algorithms
This project implements core graph algorithms used in computer science and real-world systems.

Graph Algorithms (C++)

## Description
This project implements core graph algorithms used in computer science and real-world systems.

## Algorithms Implemented
- Depth-First Search (DFS)
- Breadth-First Search (BFS)
- Dijkstra’s Algorithm (shortest path)
- Bellman-Ford Algorithm (handles negative weights)
- Graph connectivity detection

## Concepts Covered
- Graph traversal
- Shortest path optimization
- Negative cycle detection
- Adjacency list representation

## Technologies
- C++
- STL (vectors, queues)

## What I Learned
- Tradeoffs between Dijkstra and Bellman-Ford
- How graph structure impacts performance
- Practical implementation of traversal algorithms
